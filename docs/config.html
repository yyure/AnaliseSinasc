Módulo de configuração¶

Módulo de configurações

Este módulo contém as configurações para a limpeza dos dados brutos

Funcionalidades: - Contém as colunas a serem removidas. - Contém as colunas com restrições de valores. - Contém as colunas a serem filtradas por média, z-score e alguns valores especificos. - Gera arquivo yaml.

modules.config.generate_config_file(path: str)¶

Gera o arquivo de configuração para a limpeza dos dados

Parâmetros:

path (str) – Endereço em que o arquivo será gerado

Tipo de retorno:

None
