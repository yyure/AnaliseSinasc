Análise de Dados e Visualização - Henzo¶

Módulo de Análise de Dados e Visualização de Dados

Este módulo contém funções para análise de dados de saúde materna e infantil.

Funcionalidades: - Analisa os dados de peso dos bebês, cria um histograma e o salva. - Analisa os dados do índice APGAR por raça, cria um gráfico e o salva. - Analisa a quantidade de filhos mortos por raça, cria um gráfico e o salva. - Analisa o peso do bebê em relação à idade da mãe, cria um gráfico de dispersão e o salva.

modules.analysis.henzo.analysis.analise_apgar_raca(path_input: str)¶

Trabalha com os dados limpos e plota o gráfico APGARxRACA, salvando-a em ./images/.

Parâmetros:

path_input (str) – Caminho dos dados SISNASC já limpos anteriormente.

Tipo de retorno:

None

Exemplos

>>> analise_apgar_raca('data/dados.csv')
